# Agent Routing Matrix

Quick reference for selecting the right agent for your task.

## Core Development Agents

| Agent | Use When | Key Strengths |
|-------|----------|---------------|
| **Engineer** | Implementing features, fixing bugs, TDD work | Test-first development, Fortune 10 scale experience, micro-cycle iteration |
| **Architect** | System design, architectural decisions, feature specs | PhD-level distributed systems, constitutional principles, strategic planning |
| **Designer** | UX/UI design, design systems, user research | Design school training, pixel perfection, accessibility-first |
| **QATester** | Validating functionality, browser-based testing, quality gates | Gate 4 enforcement, browser-automation exclusive, evidence-based validation |

## Selection Guidelines

### Start with Intent

**Building or fixing code?** → Engineer (with QATester for validation)

**Designing architecture?** → Architect (use /plan mode for strategic decisions)

**Need visual design?** → Designer (UI/UX expertise)

**Research needed?** → Use the Research skill (it orchestrates research agents)

**Security testing?** → Pentester (authorized testing only)

**General problem-solving?** → Intern (high-agency generalist)

### Agent Combinations

**Full feature development:**
1. Architect (design)
2. Engineer (implement)
3. Designer (review UX)
4. QATester (validate)

**PRD-driven work:**
1. Architect (write spec)
2. Algorithm (generate ISC)
3. Engineer (execute)
4. QATester (verify)

**Product decisions:**
1. ProductStrategist (recommend)
2. TechnicalReviewer (evaluate)
3. StakeholderCommunicator (communicate)

## Agent Isolation

| Isolation Mode | Agents | Behavior |
|----------------|--------|----------|
| **worktree** | Engineer, Architect | Isolated git worktree per session |
| **none** | ProductStrategist, TechnicalReviewer, StakeholderCommunicator | Shared working directory |
| **default** | All others | Standard isolation |

## Important Notes

- Research agents are NOT invoked directly — use `Skill("Research")` instead
- Artist is called BY Media skill workflows — not directly
- Pentester only for authorized defensive security tasks
- BrowserAgent and UIReviewer designed for parallel execution
- All agents follow PAI output format (structured with emoji sections)
